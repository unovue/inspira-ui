#api

## API

| Nom de propriété | Type     | Valeur par défaut | Description                                  |
| ---------------- | -------- | ----------------- | -------------------------------------------- |
| `color`          | `string` | `"#FFF"`          | Couleur des étoiles dans le champ.           |
| `count`          | `number` | `200`             | Nombre d'étoiles affichées dans l'animation. |
