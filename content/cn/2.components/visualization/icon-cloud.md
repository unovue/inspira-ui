#api

## API

| 属性名 | 类型     | 默认值 | 描述                                   |
| ------ | -------- | ------ | -------------------------------------- |
| `class`| `string` | -      | 自定义组件样式的额外类名。             |
| `images`| `array` | `[]`   | 要在云中渲染的图片 URL 数组。          |
