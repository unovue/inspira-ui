## API

| 属性名  | 类型     | 默认值 | 描述                                   |
| ------- | -------- | ------ | -------------------------------------- |
| `delay` | `number` | `0.5`  | 动画开始前的延迟（秒）。               |
| `duration`| `number`| `0.8` | 动画持续时间（秒）。                   |
| `class` | `string` | `""`   | 用于自定义样式的额外 CSS 类。          |
