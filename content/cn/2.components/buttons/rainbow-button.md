#api

## API

| 属性名 | 类型     | 默认值      | 描述                                |
| ------ | -------- | ----------- | ----------------------------------- |
| `class`| `string` | `""`        | 应用于按钮的额外 CSS 类。           |
| `is`   | `string` | `"button"`  | 渲染组件时使用的 HTML 标签。        |
| `speed`| `number` | `2`         | 动画持续时间（秒）。                |
