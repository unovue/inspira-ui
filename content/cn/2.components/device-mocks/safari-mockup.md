#api

## API

| 属性名 | 类型     | 默认值 | 描述                              |
| ------ | -------- | ------ | --------------------------------- |
| `url`  | `string` | `null` | 模型地址栏中显示的 URL。          |
| `src`  | `string` | `null` | 在模型中显示的图片 URL。          |
| `width`| `number` | `1203` | 模型 SVG 的宽度（像素）。         |
| `height`| `number`| `753`  | 模型 SVG 的高度（像素）。         |
