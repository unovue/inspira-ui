## API

| Prop Name      | Type     | Default | Description                           |
| -------------- | -------- | ------- | ------------------------------------- |
| `duration`     | `number` | `10`    | Duration of the animation in seconds. |
| `radiantWidth` | `number` | `100`   | Width of the radiant animation.       |
